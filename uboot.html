Program to boot freebsd on arm devices is named "u-boot" but this section is named "Das u-boot" beacuse it is a official name for this program. For porting MeVsys kernel you must first port u-boot only if your device to port MeVsys doesn't have ported u-boot. After creating u-boot port you must create freebsd port for build u-boot in building kernel for MeVsys and create config device for MeVsys kernel. I recommend to read guide how to port u-boot https://elinux.org/images/2/2a/Schulz-how-to-support-new-board-u-boot-linux.pdf
