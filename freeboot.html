Program to boot freebsd on arm devices is named "freeboot". For porting MeVsys kernel you must first port freeboot only if your device to port MeVsys doesn't have ported freeboot. After creating freeboot port you must create freebsd port for build freeboot in building kernel for MeVsys and create config device for MeVsys kernel.
